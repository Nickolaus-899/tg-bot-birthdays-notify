# 🎂 Birthday Notification Bot

Telegram-бот на базе **aiogram 3** для уведомлений о днях рождения.

## Возможности

- Хранение дней рождения в `./data/birthdays.json`
- Подписка/отписка пользователей от уведомлений
- Ежедневные уведомления в **10:00** всем подписчикам
- Управление записями только для администраторов
- Запуск в Docker-контейнере

## Команды

| Команда | Доступ | Описание |
|---|---|---|
| `/start` | все | Приветствие |
| `/help` | все | Список команд |
| `/status` | все | Статус подписки |
| `/list` | все | Список всех дней рождения |
| `/subscribe` | все | Подписаться на уведомления |
| `/unsubscribe` | все | Отписаться от уведомлений |
| `/add` | админ | Добавить день рождения (FSM) |
| `/remove` | админ | Удалить запись по ID |
| `/cancel` | все | Отменить текущее действие |


## Структура проекта

```
.
├── data/                    # JSON-файлы с данными (создаются автоматически)
│   ├── admins.json          # список ID администраторов
│   ├── birthdays.json       # список дней рождения
│   ├── subscribers.json     # подписчики уведомлений
│   └── counter.json         # глобальный счётчик ID
├── src/
│   ├── __init__.py          # инициализация bot, dp, router
│   ├── db.py                # пути к файлам, init_db, хелперы
│   ├── storage.py           # универсальные функции работы с JSON
│   ├── notifications.py     # планировщик уведомлений (APScheduler)
│   ├── main_router.py       # регистрация всех обработчиков
│   └── handlers/
│       ├── subscribe.py     # /subscribe, /unsubscribe
│       └── birthdays.py     # /add, /remove (только для админов)
├── main.py                  # точка входа
├── Dockerfile
├── docker-compose.yml
├── requirements.txt
└── .env
```

## Формат файлов данных

**`admins.json`**
```json
[123456789, 987654321]
```

**`birthdays.json`**
```json
{
  "1": { "name": "Иван Иванов", "birthday": "05-25" },
  "2": { "name": "Мария Петрова", "birthday": "12-01" }
}
```

**`subscribers.json`**
```json
[123456789, 987654321]
```
